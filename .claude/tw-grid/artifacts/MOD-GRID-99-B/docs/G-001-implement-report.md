### ADR-MOD-GRID-99-B-001: C-5 vs Docusaurus customCss → customCss: [] 채택
- **결정**: custom.css 미생성, customCss: []
- **대안 1**: spec 그대로 custom.css 생성 → C-5 하드 제약 위반

### ADR-MOD-GRID-99-B-002: Docusaurus v3 채택 (Nextra 미채택)
- **결정**: @docusaurus/core ^3.0.0 (MIT)
- **대안 1**: Nextra — Next.js 의존성 과도, TypeDoc 플러그인 성숙도 낮음
- **대안 2**: VitePress — Vue 기반, 팀 React 친화성 낮음
- **trade-off**: 생태계 친화성 vs 의존성 추가

### ADR-MOD-GRID-99-B-003: docusaurus-plugin-typedoc 채택 (수동 MDX 미채택)
- **결정**: docusaurus-plugin-typedoc ^1.0.0 (MIT) + typedoc ^0.27.0 (Apache-2.0)
- **대안 1**: 수동 MDX 변환 — 유지 비용 과도
- **대안 2**: TypeDoc HTML iframe — 검색/테마 통합 불가
- **trade-off**: 자동화 편의성 vs 출력 제어 범위

### ADR-MOD-GRID-99-B-004: 단일 로케일 ko (G-005 위임)
- **결정**: defaultLocale: 'ko', locales: ['ko']
- **대안 1**: ko + en 동시 설정 — Goal AC 범위 초과
