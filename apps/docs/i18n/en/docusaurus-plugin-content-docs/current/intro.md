# topgrid

**A headless React grid solution built on TanStack Table.** It ships as modular
packages spanning everything from core features such as display/edit cell renderers,
filtering, sorting, pagination, and virtualization, to Enterprise-grade Pro features
such as pivoting, server-side row model (SSRM), integrated charts, spreadsheet,
advanced filters, range selection, aggregation, and cell merging.

- npm scope: `@topgrid` · **31 packages** (7 free MIT + 24 commercial EULA)
- License: MIT core/adapters + commercial Pro packages
