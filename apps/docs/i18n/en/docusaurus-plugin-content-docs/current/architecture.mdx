It is divided into a React core + Vue adapter (multi-framework), Pro features, and a chart family.

- **Every Pro package в†’ `@topgrid/grid-license`** (runtime license gate).
- **Most Pro в†’ `@topgrid/grid-core`** (built on top of `<Grid>`).
- Some Pro packages **compose other Pro/Open packages**: `grid-pro-pivot`В·`grid-pro-panel` в†’ `grid-pro-agg` В· `grid-pro-edit-plus` в†’ `grid-pro-tracking` В· `grid-pro-filter` в†’ `grid-features` В· `grid-pro-sheet` в†’ `grid-pro-range` В· `grid-pro-master` в†’ `grid-export`.
- Internal dependencies use `workspace:*` в†’ exact-version pins on publish (lockstep). The `@topgrid/grid` facade re-exports every package.

## Pro Package License Activation

When using Pro packages, a runtime license check runs via `@topgrid/grid-license`.
Set your license key once at the app entry point.

```tsx
import { setLicenseKey } from '@topgrid/grid-license';

setLicenseKey('YOUR-LICENSE-KEY');
```
