# 시작하기

topgrid는 TanStack Table v8 기반의 headless React 그리드 라이브러리입니다.
21개 패키지로 구성된 monorepo로, 공개(MIT) 코어 + Pro 기능을 모두 제공합니다.

## 설치

```bash
# 공개 패키지 (headless core)
pnpm add @topgrid/grid @topgrid/grid-core @topgrid/grid-features \
         @topgrid/grid-renderers @topgrid/grid-export @topgrid/grid-license

# peer dependencies
pnpm add react react-dom @tanstack/react-table @tanstack/react-virtual
```

Pro 패키지(피벗·서버사이드·차트·스프레드시트·고급 필터·편집·집계 등)는 라이선스가 필요합니다 —
[아키텍처](./architecture)의 패키지 목록과 [기능 비교](./comparison)를 참고하세요.

```bash
# 예: 필요한 Pro 패키지만 선택 설치
pnpm add @topgrid/grid-pro-pivot @topgrid/grid-pro-serverside @topgrid/grid-pro-chart
```

## 기본 사용 예시

`<Grid>`는 `columns`/`data`를 직접 받습니다(headless — TanStack `ColumnDef` 그대로 사용).

```tsx
import { Grid } from '@topgrid/grid';
import type { ColumnDef } from '@tanstack/react-table';

interface Person {
  id: number;
  name: string;
}

const columns: ColumnDef<Person>[] = [
  { accessorKey: 'id', header: 'ID' },
  { accessorKey: 'name', header: '이름' },
];

const data: Person[] = [
  { id: 1, name: '홍길동' },
  { id: 2, name: '김철수' },
];

export function BasicGrid() {
  return <Grid<Person> columns={columns} data={data} enableSort enableFilter />;
}
```

정렬·필터·페이지네이션·선택 등은 `enableSort` / `enableFilter` / `enablePagination` 같은
플래그로 켭니다. 상태를 외부에서 제어하려면 `@topgrid/grid-core`의 `useGridState` 훅으로
controlled state(`onSortingChange` 등)를 배선할 수 있습니다.
