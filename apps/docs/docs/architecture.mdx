# 아키텍처

TOMIS Grid는 13개 패키지로 구성된 monorepo 구조입니다.
공개(MIT) 패키지와 Pro(상용 라이선스) 패키지로 분류됩니다.

## 패키지 목록

| 패키지 | 분류 | 설명 |
|--------|------|------|
| `@tomis/grid` | Open (MIT) | 메타 패키지 — 전체 공개 패키지 re-export |
| `@tomis/grid-core` | Open (MIT) | TanStack Table 추상화 wrapper + `useGridState` 핵심 hook |
| `@tomis/grid-features` | Open (MIT) | 정렬·필터·페이지네이션 등 공통 기능 빌더 옵션 |
| `@tomis/grid-export` | Open (MIT) | Excel/CSV/PDF 내보내기 (`xlsx`, `jspdf` peer) |
| `@tomis/grid-renderers` | Open (MIT) | 셀 렌더러 컴포넌트 모음 (날짜·숫자·배지 등) |
| `@tomis/grid-license` | Open (MIT) | 런타임 Pro 라이선스 검증 모듈 |
| `@tomis/grid-pro-agg` | Pro | 집계(Aggregation) — 그룹별 합계·평균·카운트 |
| `@tomis/grid-pro-datamap` | Pro | 데이터 매핑 — 코드→라벨 변환, 계층형 셀렉트 |
| `@tomis/grid-pro-header` | Pro | 고급 헤더 — 다중 헤더 병합, 고정 헤더 |
| `@tomis/grid-pro-master` | Pro | 마스터-디테일 — 행 확장 + 중첩 그리드 |
| `@tomis/grid-pro-merging` | Pro | 셀 병합 — 동일 값 자동 rowSpan/colSpan |
| `@tomis/grid-pro-range` | Pro | 범위 선택 — 드래그 셀 선택·복사·붙여넣기 |
| `@tomis/grid-pro-tracking` | Pro | 변경 추적 — 편집 행·셀 dirty 상태 관리 |

## Pro 패키지 라이선스 활성화

Pro 패키지 사용 시 `@tomis/grid-license`를 통해 런타임 라이선스 검증이 실행됩니다.
앱 진입점에서 라이선스 키를 한 번 초기화하세요.

```tsx
import { initLicense } from '@tomis/grid-license';

initLicense('YOUR-LICENSE-KEY');
```
