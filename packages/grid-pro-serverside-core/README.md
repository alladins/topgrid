# @topgrid/grid-pro-serverside-core

프레임워크 무관 서버사이드 로우 모델 코어 — 블록 캐시(SSRM), 데이터 흐름 컨트롤러,
지연 그룹 트리 캐시, 뷰포트 로우 모델(push), 서버 피벗 컬럼 파생. React/Vue 의존 0
(`@tanstack/table-core` 는 타입 전용).

React 결합 훅(`@topgrid/grid-pro-serverside`)과 Vue(`@topgrid/grid-pro-serverside-vue`)가
공유 소비한다. 상용 라이선스(EULA). 라이선스 게이트는 소비 표면에서 수행(코어엔 없음).
